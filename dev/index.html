Goal

The goal of the package JToric is to make computations on toric geometry accessible in the OSCAR computer algebra system. As such, we follow the book Toric Varieties by David A. Cox, John B. Little and Henry K. Schenck]. Currently, this project is work-in-progress. Our goal is to ensure that one can perform all computations of Appendix B.

Generalities

JToric.version — Constant
version

The version number of the loaded JToric. Please mention this number in any bug report.

JToric.create_divisor — Function
create_divisor( c::Vector{Int}, v::NormalToricVariety )

Construct the torus invariant divisor on the normal toric variety v as linear combination of the torus invariant prime divisors of v. The coefficients of thi linear combination are passed as list of integers as first argument.

Examples

julia> create_divisor( [1,1,2], projective_space( 2 ) )
toric_divisor(GAP: <A divisor of a toric variety with coordinates ( 1, 1, 2 )>)
